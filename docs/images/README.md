# Project Images

This directory contains images used in the main README.md and project documentation.

## Required Images

The main README.md references the following images:

- `motion-shell-assembled.jpg` - Photo of the assembled motion shell
- `carriage-module-exploded.jpg` - Exploded view of the carriage module
- `sensor-board.jpg` - Photo of the sensor board

## Additional Images

Feel free to add:
- Build progress photos
- Assembly documentation images
- Test setup images
- Oscilloscope captures
- CAD renders

## Image Guidelines

- Use descriptive filenames (e.g., `pcb-slip-ring-assembly.jpg`)
- Recommended formats: JPG for photos, PNG for diagrams/screenshots
- Keep individual images under 2MB when possible
- Use reasonable resolution (1920px width max for photos)
